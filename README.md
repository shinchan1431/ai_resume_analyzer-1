# AI Resume Analyzer 🚀

An intelligent resume analyzer built using **Python + Streamlit** that extracts skills from resumes and provides:

* Skill detection
* Career field prediction
* Resume scoring
* Job role matching
* Live job search using Adzuna API
* Interview preparation questions
* Skill gap analysis
* Course recommendations
* Visualization charts

---

## 🔧 Technologies Used

* Python
* Streamlit
* PyPDF2
* Pandas
* Matplotlib
* Adzuna Job API

---

## 📊 Features

### Resume Upload

Upload PDF resumes and extract text automatically

### Skill Detection

Identifies technical and soft skills from resume

### Career Prediction

Suggests best matching career domain

### Resume Score

Calculates readiness score out of 100

### Live Job Suggestions

Fetches real-time jobs using API

### Interview Questions

Provides role-based interview preparation questions

### Skill Gap Analysis

Shows missing skills required for selected job role

---

## ▶️ Run Locally

pip install -r requirements.txt

Run app

streamlit run app.py
