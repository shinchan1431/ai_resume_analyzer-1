# ai_resume_analyzer-1
AI Resume Analyzer is a Streamlit-based web application that analyzes resumes and provides useful insights such as resume score, detected skills, and job recommendations.

## Features

- Upload resume (PDF format)
- Extract skills automatically
- Generate resume score
- Suggest improvements
- Recommend job roles based on skills
- Graph visualization of skill analysis

## Technologies Used

- Python
- Streamlit
- NLP (Natural Language Processing)
- Machine Learning

## How to Run the Project

Step 1:
Clone the repository

git clone https://github.com/shinchan1431/ai-resume-analyzer.git

Step 2:
Install dependencies

pip install -r requirements.txt

Step 3:
Run the application

streamlit run app.py

## Project Objective

This project helps students and job seekers:

- Understand their resume strength
- Identify missing skills
- Improve job readiness
- Get AI-based career suggestions
